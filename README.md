RevShop – E-Commerce Web Application

Description:

RevShop is a full-stack e-commerce web application developed using Java, Spring Boot, Spring Security, and Oracle Database.
The platform supports buyers and sellers, allowing users to browse products, manage carts, place orders, and sellers to manage inventory.

Features:

User registration and login

Browse and search products

Add products to cart

Place orders and checkout

Seller product management

Order tracking

Product reviews and wishlist

Secure authentication with Spring Security

Technology Stack

Backend: Java, Spring Boot

Security: Spring Security

Database: Oracle DB

ORM: Spring Data JPA

Frontend: HTML, CSS, JavaScript

Build Tool: Maven

Architecture:
Client (Browser)
      ↓
Controller Layer
      ↓
Service Layer
      ↓
Repository Layer
      ↓
Oracle Database

Security:

Role-based authentication

Password encryption (BCrypt)

Session-based login with Spring Security
